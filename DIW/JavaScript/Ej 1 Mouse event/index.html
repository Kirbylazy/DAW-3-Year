Ejercicio 1 — Eventos del ratón

Dario Aguilar Rodriguez

click

Se produce cuando el usuario pulsa y suelta el botón principal del ratón sobre un elemento.

dblclick

Se ejecuta cuando el usuario realiza dos clics seguidos sobre el mismo elemento.

mousedown

Se activa en el momento en el que el usuario presiona un botón del ratón.

mouseup

Ocurre cuando el usuario suelta el botón del ratón.

mouseenter

Se lanza cuando el puntero del ratón entra en el área del elemento.

mouseleave

Se produce cuando el puntero del ratón sale del área del elemento.
